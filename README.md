# AI MANTRAAS Coming Soon Site

Premium animated marketing site built with Next.js, TailwindCSS, GSAP, and Framer Motion.

## Pages

- `/` - Coming Soon landing page with countdown, email registration, and social CTAs
- `/program` - 15-Day Automation Learning Program
- `/products` - Product showcase including AI xSDR

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
